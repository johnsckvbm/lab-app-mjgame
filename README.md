# lab-app-mjgame

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面均为自包含的前端文件，不依赖于特定域名或后端服务，适合直接通过浏览器打开或托管于静态文件服务中。

## 目录说明

```
lab-app-mjgame/
├── README.md        # 本文件
└── pages/           # 存放独立 HTML 页面文件
    └── ...
```

- `pages/`：所有归档的 HTML 页面均放置在此目录下，每个文件为一个独立页面。

## 页面归档说明

- 每个 HTML 页面均为功能完整的独立文件，包含所需的样式与脚本。
- 页面不绑定任何特定域名，可自由部署至任意静态托管平台。
- 归档内容仅供学习、参考或存档用途，不针对任何特定网站或服务。

## 使用方式

1. 克隆或下载本仓库。
2. 使用浏览器直接打开 `pages/` 下的任意 HTML 文件。
3. 或将 `pages/` 目录部署至静态服务器，通过 URL 访问对应页面。

## 维护说明

- 新增页面时，请将 HTML 文件放入 `pages/` 目录，并保持文件命名清晰简洁。
- 若需更新现有页面，请直接替换对应文件，并确保功能完整性。
- 不建议在页面中引入外部资源（除非必要），以保持页面自包含特性。

## 贡献

欢迎提交 Pull Request 或 Issue 来改进页面或修复问题。提交前请确保文件不包含敏感信息或指向特定域名的链接。
